Инструкция к использованию

Управление контентом осуществляется посредством меню слева:

  • Предложить = написать в радиостудию. Вы прислать реквест, предложить какой-то материал к эфиру. Идеи по поводу улучшения этого конкретного проекта (RAM) лучше предлагать лично автору, используя контакты со страницы Гитхаба.
  • Жанры = список тегов и тематических направлений кинематографа, доступных на данный момент.
  • Случайный фильм. И так всё понятно.
  • Настройки = настройки отображения и интерактива.
  • Поиск = поиск по базе данных по ключевым словам, разделённым пробелом (запятые ставить не надо). Ищется по описанию, названию, жанрам.
  • Сброс = сброс результатов поиска, подгруженных результатов и прочего (возврат к исходному состоянию страницы).

Подробнее о настройках:

  • Лимит подгрузки = ограничение на количество элементов, загружаемых при заходе, при подгрузке по клику MOAR и при поиске.
  • Тема = цветовая схема оформления на выбор, какая более приятна вашему глазу в ваше время суток - тёмная или светлая (автоматически = определять тему по системным часам - ночью тёмная, днём светлая).
  • Поиск = режим поиска (работает пока только редуцирующий, и неизвестно, будут ли остальные, ибо это никому особо не нужно, но для интереса пусть будут).
    • Редуцирующий - поиск, итеративно отсекающий от полного списка БД элементы, не содержащие кейворд (таким образом, очерёдность ключевых слов в запросе имеет значение приоритетов).
    • Спиральный - поиск, работающий по принципу конъюнктивного, но пирамидально - сначала он выбирает пересечение множеств всех ключевых слов, затем пересечения, содержащие не все слова (хотя бы 2 из 3, затем хотя бы 1 из 3 и т.п.). Таким образом, сверху - самые подходящие результаты, снизу - меньше всего.
    • Конъюнктивный - поиск по пересечению множеств, образованных каждым из ключевых слов. Выбирается пересечение, содержащее строго все ключевые слова.
    • Дизъюнктивный - поиск жадного типа, показывающий все результаты, которые содержат хотя бы одно из ключевых слов.
    • Ассоциативный - поиск итеративно-анализирующего типа, выбирающий полные множества по каждому результату и реагирующий на совпадения (в идеале какое-то слово должно присутствовать во всех).
  • Рандом = количество элементов, случайно выбираемых по нажатию на кнопку "случайная кассета".
  • Радио = настройка состояния радио по умолчанию (убрано = вовсе отсутствует на странице).
  • Debug = режим дебага для разработчиков (показываются полные сообщения об ошибках, идёт отладочная информация в консоль).
